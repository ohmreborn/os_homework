สำหรับ code ต่อไปนี้ จะสามารถ รันได้อย่างถูกต้องสำหรับ ระบบปฎิบัติการ macos เท่านั้น กรณีที่ใช่ ระบบปฎิบัติการอื่นอาจจะไม่สามารถ make ไฟล์ขึ้นมาได้ หรือ code รันออกมาได้ผลลัพธ์ไม่ตรงกับในรายงาน

# compile code
```
make
```

# run code
โดยสามารถ recompile ให้โดยอัตโนมัติ โดยไม่ต้อง พิมพ์ว่า make ใหม่อีกรอบ
```
make run
```
สามารถใส่ขนาดของ memory ที่อยากจะจองได้ เช่น 60MB ให้ใส่คำว่า MEM=60 ต่อท้าย
```
make run MEM=60
```

กรณี อยากลองทสอบ memory หลายครั้ง ให้ใช้ script ที่เตรียมไว้ให้ 
```
./run.sh
```
